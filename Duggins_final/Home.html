About Gaming Computers


Tradition since 1951

The first gaming computer was made in 1951 by John Makepeace Bennett. It was called the Nimrod and wasn't intended to be a gaming computer, it was meant to solve complex mathematical equations but people ened up using it for fun.

A gaming computer is a computer that is made specifically to play video games. These computers usually have high end parts that are very powerful so they can run any game.

The Parts of a Gaming PC


CPU

The Central Processing Unit completes the commands that the computer gives it.


GPU

The Graphics Processing Unit speeds up the time it takes for the computer to process and create images


RAM

Random Access Memory gives applications a short-term place to store things that don't need to permanently be downloaded.


Storage

HDDs and SSDs are ways to store information on a computer.


Cooling

Liquid and air cooling need to move hot air out and cool air in so parts don't overheat.


PSU

The Power Supply Unit supplies power to all the components so the PC works.


OS

A computer needs and operating system with a good UI so users can do things without needing to know binary.


Monitor

The monitor displays what the computer needs it to in crisp resolution.


Mouse, Keyboard, and Other Accessories

The Mouse and Keyboard are just one of many input methods users use to navigate the OS.
